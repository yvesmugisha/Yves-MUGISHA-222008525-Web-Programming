# Personal Web Programming Projects

This repository contains a set of academic web programming projects developed as part of coursework requirements. The projects focus on basic web development concepts using standard web technologies.

## Overview

The repository includes four separate projects, each demonstrating different aspects of web development such as page structure, forms, styling, scripting, and database interaction.

## Projects

### Project 1: Basic Website Layout

A simple website built using HTML and CSS. It includes a navigation bar and structured content sections based on assignment requirements.

### Project 2: Student Registration System

A registration form connected to a MySQL database using PHP. It allows storing student details submitted through the form.

### Project 3: Hotel Website

A multi-page hotel website that includes:

* Home page and additional content pages
* Order form and contact form
* User login and logout functionality
* Database connection using PHP and MySQL

### Project 4: Currency Converter

A simple currency converter built using HTML, CSS, and JavaScript for real-time conversion functionality.

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL

## Purpose

This repository is created for academic learning and submission purposes, focusing on fundamental web development skills.
